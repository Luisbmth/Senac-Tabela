Formatação de Texto no HTML

Em HTML, é possível aplicar diferentes tipos de formatação ao texto usando tags específicas:

  • - <b> → deixa o texto em negrito;
  • - <i> → deixa o texto em itálico;
  • - <u> → sublinha o texto;
  • - <strong> → indica que o texto tem importância;
  • - <em> → indica ênfase;
  • - <mark> → destaca o texto com marcação.
